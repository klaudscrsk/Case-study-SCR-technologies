SCR TECHNOLOGIES — MARKETINGOVE PODKLADY
========================================

Ahoj, toto je vstupny balik pre marketing SCR technologies.
Su to PODKLADY (surovina) — fakty, positioning, produkty, ton, referencie.
Nie su to hotove texty. Hotove posty / case studies / web copy si z toho
vyrobis sama cez Claude Code.

PODMIENKA
---------
Potrebujes mat nainstalovany Claude Code (aplikacia alebo terminal) a prihlaseny
ucet. Nic ine — ziadny server, Node ani build. Tento balik su len subory.

AKO ZACAT (3 kroky)
-------------------
1. ROZBAL zip do normalneho priecinka, kde ho najdes — najlepsie Dokumenty alebo
   Plocha (napr. Dokumenty/scr-marketing-pack/).
   - NIE do Downloads (lahko sa straci) ani do zdielaneho cloudu.
   - Cesta idealne bez diakritiky a medzier (menej problemov).
2. Otvor Claude Code V TOMTO priecinku:
   - Aplikacia: Open folder -> vyber rozbaleny priecinok.
   - Terminal:  cd cesta/k/priecinku  a spusti  claude
   Claude si automaticky nacita subor CLAUDE.md a bude mat cely kontext.
3. Napis mu napr.: "Precitaj si vsetky podklady a navrhni 3 linkedin posty
   o produkte Digitalne kanaly." A ides.

CO MU NAPISAT (rychle frazy)
----------------------------
- LinkedIn posty:    "Daj 3 linkedin posty o [produkte/referencii] z podkladov"
- Case study:        "Vygeneruj case study pre [KLIENT], podklady som dala do inputs/"
- Produktova stranka: "Vygeneruj produktovu stranku pre [PRODUKT], podklady su v inputs/"
- Web copy:          "Napis text na produktovu stranku [PRODUKT]: co riesi, komu, co prinasa, CTA"

Tip: podklady (Word/PDF/poznamky/obrazky) najprv hod do priecinka inputs/.
Cim viac das (cisla, citacie, painpointy), tym lepsie. Co Claude nema, nevymysli
— povie ti, co chyba. Vystup (HTML) si otvor dvojklikom v prehliadaci.

CO JE V BALIKU
--------------
CLAUDE.md                        - mozog: kontext + pravidla pre Claude (cita sa automaticky)
01_firma_a_positioning.md        - kto je SCR (z intro decku): positioning, vizia, segmenty, klienti
02_produkty_podklady.md          - 9 produktov: tagline, painpointy, prinos, klucove slova, klienti
03_referencie.md                 - kniznica referencii: plne popisy projektov, cisla, citacie

Salesdecky a onepagery k produktom nie su v tomto balíku — stiahnes si ich
